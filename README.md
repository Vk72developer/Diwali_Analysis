🪔 Diwali Sales Data Analysis

📌 Project Overview

This project analyzes Diwali sales data using Python to uncover customer purchasing behavior, sales trends, and business insights. The analysis helps identify high-value customers, popular product categories, and demographic patterns that can support data-driven marketing and business decisions.

🎯 Objectives

* Clean and preprocess the dataset
* Perform Exploratory Data Analysis (EDA)
* Analyze customer demographics
* Identify top-selling products and categories
* Discover purchasing patterns based on age, gender, occupation, and state
* Generate actionable business insights through data visualization

🛠️ Tech Stack

* Python
* Jupyter Notebook
* NumPy
* Pandas
* Matplotlib
* Seaborn

📊 Key Analysis

* Data Cleaning & Preprocessing
* Handling Missing Values
* Exploratory Data Analysis (EDA)
* Gender-wise Purchase Analysis
* Age Group Analysis
* State-wise Sales Performance
* Marital Status Analysis
* Occupation-wise Spending
* Product Category Analysis
* Data Visualization

📈 Key Insights

* Identified the customer segments contributing the highest sales.
* Analyzed purchasing behavior across different age groups and genders.
* Determined the top-performing states and occupations.
* Identified the most popular product categories during the Diwali season.
* Generated visual insights to support business and marketing strategies.
